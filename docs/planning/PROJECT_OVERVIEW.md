# Project Overview

## Vision

The TCP Congestion Control Visualizer is an interactive educational simulator designed to help users understand how TCP congestion control algorithms respond to changing network conditions.

## Goals

- Build a production-quality networking simulator.
- Visualize packet transmission and acknowledgements.
- Compare multiple TCP congestion control algorithms.
- Display congestion window growth in real time.
- Provide statistical analysis of network performance.

## Target Users

- Computer Science students
- Networking learners
- Software engineers
- Interview candidates
- Educators

## Supported Algorithms

- TCP Tahoe
- TCP Reno
- TCP New Reno
- TCP Cubic

Future support:
- BBR

## Deliverables

- Interactive web-based simulator
- Real-time congestion window graphs
- Packet transmission animations
- Performance comparison dashboard
- GitHub-ready documentation
