# Resume Description

### TCP Congestion Control Visualizer | Python, FastAPI, React, TypeScript

- Built an interactive TCP congestion control simulator supporting Tahoe, Reno, New Reno, and Cubic algorithms.
- Implemented packet-level network simulation with configurable latency, bandwidth, and packet loss.
- Developed real-time visualizations for congestion window growth, throughput, RTT, and packet retransmissions.
- Designed a full-stack architecture using FastAPI for simulation APIs and React for an interactive dashboard.
- Performed comparative analysis of congestion control algorithms under varying network conditions.
